Aluminum CSS

Aluminum CSS is a lightweight mini CSS framework that is easy to use and customize.

Margins, paddings, gaps

Aluminum CSS uses a consistent spacing system that ensures good readability and visual balance. There are three sizes: mini, standard and large. The utilities are similar to Tailwind CSS.

Input

Input is a text field element 

<input class="alu" type="text" placeholder="Enter your name" />

Textarea

Textarea is a multi-line text field element 

<textarea class="alu" placeholder="Enter your message"></textarea>

Checkbox

Checkbox is a checkbox element 

<input class="alu" type="checkbox" />

Radio

Radio is a radio button element 

<input class="alu" type="radio" />

Range

Range is a slider element 

<input class="alu" type="range" />

Button

Button is a clickable element

<button class="alu">Secondary</button>

Dialog

Dialog is a modal dialog element 

<dialog class="alu" style="min-width: 400px;">
  <h5 class="alu mv-s mb-l">
    Add new item
  </h5>
  <input class="alu mb-s" type="text" placeholder="Enter item name" />
  <div class="alu inverted-row gap-s">
    <button class="alu primary" onclick="document.getElementById('dialog-example').close()">Add</button>
    <button class="alu" onclick="document.getElementById('dialog-example').close()">Cancel</button>
  </div>
</dialog>
<button class="alu primary" onclick="document.getElementById('dialog-example').showModal()">Open dialog</button>
