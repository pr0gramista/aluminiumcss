Aluminum CSS

Aluminum CSS is a lightweight mini CSS framework that is easy to use and customize.

Margins, paddings, gaps

Aluminum CSS uses a consistent spacing system that ensures good readability and visual balance. There are three sizes: mini, standard and large. The utilities are similar to Tailwind CSS.

Selective styling

Aluminium CSS is designed to be easily used with other CSS styles. You can apply it globally by putting the .alu class on the body element, or you can use it selectively by adding the class to specific elements.

If you want to remove styling from certain elements you can apply .notalu to the elements or their parent. 

<div class="column gap-md">
  <div>
    <button>Styled via parent</button>
  </div>
  <div>
    <button class="alu">Styled directly</button>
  </div>
  <div class="notalu">
    <button>Unstyled via parent</button>
  </div>
  <div>
    <button class="notalu">Unstyled directly</button>
  </div>
</div>
