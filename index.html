Aluminum CSS

Aluminum CSS is a lightweight mini CSS framework that is easy to use and customize.

Installation

Aluminum CSS is available as a NPM package or a single file served via CDN. 

npm install aluminiumcss
// In your JS file with a bundler
require('aluminiumcss/dist/aluminium.min.css')

To use the CDN version, add the following script tag to your head HTML element:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/aluminiumcss/dist/aluminium.min.css">

Customization

Aluminum CSS is easily customizable via a selection of CSS variables.

Margins, paddings, gaps

Aluminum CSS uses a consistent spacing system that ensures good readability and visual balance. There are four sizes: sm, md, lg, xl. To apply margins, paddings, gaps you can use Tailwind CSS like classes.

Positioning

Aluminum CSS provides a set of positioning utilities. You can use the classes fixed, absolute, relative, sticky, static, top-[0/sm/md/lg/xl], right-[0/sm/md/lg/xl], bottom-[0/sm/md/lg/xl], left-[0/sm/md/lg/xl] to position elements.

top-0, left-0
top-xl, left-xl
bottom-lg right-sm
<div class="relative" style="height: 200px; width: 200px; background-color: lightgray">
  <div class="absolute debug top-0 left-0">top-0, left-0</div>
  <div class="absolute debug top-xl left-xl">top-xl, left-xl</div>
  <div class="absolute debug bottom-lg right-sm">bottom-lg right-sm</div>
</div>

Selective styling

Aluminium CSS is designed to be easily used with other CSS styles. You can apply it globally by putting the .alu class on the body element, or you can use it selectively by adding the class to specific elements.

If you want to remove styling from certain elements you can apply .notalu to the elements or their parent. 

<div class="column gap-md">
  <div>
    <button>Styled via parent</button>
  </div>
  <div>
    <button class="alu">Styled directly</button>
  </div>
  <div class="notalu">
    <button>Unstyled via parent</button>
  </div>
  <div>
    <button class="notalu">Unstyled directly</button>
  </div>
</div>
